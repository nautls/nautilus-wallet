## Description

Nautilus Wallet is a comunity driven wallet designed for Ergo Platform with focus on privacy, security and user experience.

- Easy to use
- Blazing fast
- Sandboxed NFT Gallery
- Ledger support
- Multi-wallet support
- dApp Connector

## Project setup

```

```

```bash
# Mainnet
npm run dev:mainnet
```

```bash
# Testnet
npm run dev:testnet
```

